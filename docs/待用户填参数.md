# 待用户后填参数

> 云端 Claude 自主推进 P0→P5 期间，凡规格 §7 列出的「具体内容」参数均用下表默认模板占位实现，
> 游戏可完整运行/导出。用户后填具体数值/卡牌/立绘/关卡时，按本表定位修改点即可。
> 对应规格：`docs/正式版构建规格.md` §7。

| 参数 | 默认占位值 | 实现位置 |
|---|---|---|
| 整备牌模板 | 占2·抽2·不弃·回手 | `Core/CardDef.cs` `PrepCardTemplate`、`Data/CharacterTemplates.cs` |
| 单人模式角色数 | 4（控全部） | `Data/CharacterTemplates.cs` `All()` |
| 血量上限下调速率 | 按受伤 20% 下调上限 | `Core/HealthSystem.cs` `MaxHpReductionRate` |
| 休息恢复量 | 回 25% 基础上限（不超上限） | `Core/HealthSystem.cs` `RestRestoreFraction`、`Rest()` |
| 卡组规模 | 12–18 | `Roguelike/RunState.cs` `DeckMin/DeckMax` |
| 地图层数 / Boss 数 | 3 层 + 1 Boss | `Roguelike/RunState.cs` `MapLayers`、`MapGenerator.Generate` |
| 商店货币与价格 | 金币；牌50/遗物150/移牌30/恢复上限40 | `Roguelike/Shell.cs` `Shop.Price*` |
| 4 角色定位与专属卡池 | 输出/布防/控制/治疗，各 3 张占位牌 | `Data/CharacterTemplates.cs` |
| 敌人种类 / Boss 编排 | 2 种普通敌（斩/突）+ 1 Boss 占位 | `Roguelike/MapGenerator`、`Main.cs` 示例战场 |
| 美术资源 | 占位色块/简单帧（立绘状态机已留接口） | `Views/PortraitController.cs`、`Views/GameView.cs` |
